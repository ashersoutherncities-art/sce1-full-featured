# Deal Flow Operations Guide

## Quick Reference for Managing the 3-Tier Offering

---

## Bronze Tier - Deal Flow Access ($297/month)

### What You Provide:
- [ ] Email alerts when new deals are sourced
- [ ] Access to deal dashboard (web portal)
- [ ] Basic property data for each deal:
  - Property address
  - Asking price
  - Bed/bath count
  - Square footage
  - Property type
  - ARV estimate (optional basic calc)

### Operations Needed:
1. **Deal Dashboard** - Build simple web portal or use:
   - Google Sheets with view-only link
   - Notion database
   - Airtable base
   - Custom dashboard (future)

2. **Email Automation** - When new deal added:
   - Send automated email to all Bronze subscribers
   - Include property basics + dashboard link
   - Use: Mailchimp, ConvertKit, or custom script

3. **Subscription Management**:
   - Stripe recurring billing
   - Monthly charge on same day
   - Cancel = remove dashboard access

### Fulfillment Checklist:
- [ ] New deal sourced → Add to dashboard
- [ ] Trigger email to Bronze subscribers
- [ ] Track which deals get claimed
- [ ] Monthly billing via Stripe

---

## Gold Tier - Full Analysis Package ($997/deal)

### What You Provide:
- [ ] Everything in Bronze
- [ ] Complete deal analysis report including:
  - ARV calculation with 3-5 comps
  - Detailed rehab estimate (room by room)
  - Profit projections (multiple scenarios)
  - Cash flow analysis
  - Risk assessment
  - Exit strategy recommendations (flip, BRRRR, rental)
  - Financing recommendations
  - Market trend analysis for area

### Operations Needed:
1. **Analysis Template** - Create standardized report format:
   - PDF or Google Docs template
   - Consistent sections
   - Professional branded design
   - 10-15 page deliverable

2. **Data Sources**:
   - MLS access for comps
   - ATTOM API for property data
   - Contractor estimates for rehab
   - Lender relationships for financing

3. **Turnaround Time**: 3-5 business days

### Fulfillment Checklist:
- [ ] Customer selects property → Receive address
- [ ] Pull comps (3-5 recent sales)
- [ ] Calculate ARV
- [ ] Get rehab estimate (use Cost Estimating service)
- [ ] Build profit scenarios (conservative, moderate, aggressive)
- [ ] Research financing options
- [ ] Analyze exit strategies
- [ ] Complete risk assessment
- [ ] Package report + deliver via email
- [ ] One round of Q&A included

---

## Platinum Tier - Turnkey Execution (Custom Quote)

### What You Provide:
- [ ] Everything in Gold
- [ ] Full project management for rehab:
  - Contractor sourcing & vetting
  - Bid collection & negotiation
  - Scope of work development
  - Schedule management
  - Quality control inspections
  - Budget tracking
  - Change order management
  - Weekly progress reports
  - Final walkthrough

### Operations Needed:
1. **Contractor Network**:
   - Pre-vetted subs in each trade
   - Backup contractors for each specialty
   - Insurance verification process

2. **Project Management System**:
   - Buildertrend, CoConstruct, or similar
   - Weekly photo documentation
   - Budget vs. actual tracking
   - Change order process

3. **Pricing Model**:
   - Option A: % of total project cost (12-18%)
   - Option B: Fixed fee + % over budget
   - Option C: Flat monthly retainer during project
   - Minimum project size: $50K+ rehab

### Fulfillment Checklist:
- [ ] Initial consultation → Scope property
- [ ] Create detailed SOW
- [ ] Provide custom quote
- [ ] Contract signed → Assign PM
- [ ] Source 3 bids per trade
- [ ] Client approves contractors
- [ ] Kickoff meeting
- [ ] Weekly site visits
- [ ] Weekly report to client (photos, budget, timeline)
- [ ] Address issues/changes immediately
- [ ] Final walkthrough
- [ ] Warranty follow-up (30/60/90 days)

## Pricing Strategy Notes

### Why $297 for Bronze?
- Lower than old $497 = easier entry
- Recurring revenue base
- Qualifies serious investors
- Filters tire-kickers

### Why $997 for Gold?
- Aligns with existing Deal Analysis top tier
- High enough to be premium
- Low enough to not require approval
- One deal pays for 3+ months of Bronze

### Why Custom for Platinum?
- Projects vary wildly ($50K-$500K+)
- Allows pricing flexibility
- Filters price shoppers
- Anchors value perception high

## FAQs to Prepare For

**Q: What if I'm already a Bronze subscriber and want a Gold analysis?**
A: Bronze subscribers get $200 off Gold tier ($797 vs $997)

**Q: Can I cancel Bronze anytime?**
A: Yes, cancel anytime. Access continues until end of billing period.

**Q: What's the turnaround for Gold analysis?**
A: 3-5 business days from receiving property details.

**Q: What size projects do you handle for Platinum?**
A: Minimum $50K rehab budget. No maximum.

**Q: What if the project goes over budget?**
A: We provide detailed budget tracking and require approval for any change orders. Our fee structure incentivizes staying on budget.

**Q: Do you source the deals or do I bring my own?**
A: Bronze = We source. Gold/Platinum = You can bring your own OR choose from our flow.

## Red Flags to Watch For

### Bronze Tier:
- Subscriber never clicks deal alerts (inactive)
- Rapid cancellation (<2 months)
- Complaints about deal quality

### Gold Tier:
- Customer wants analysis on junk deals
- Unrealistic ARV expectations
- Won't accept reality of analysis

### Platinum Tier:
- Undercapitalized investor
- Unrealistic timeline expectations
- Poor communication
- Wants to micromanage every decision
