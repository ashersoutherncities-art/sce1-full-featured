## Troubleshooting Common Issues

### Search Not Working
- Check JavaScript console for errors
- Verify `searchBar` element ID matches script
- Ensure array `articles` is properly defined

### Navigation Broken on Articles
- Verify `../load-header.js` path is correct
- Check that `header.html` is in root directory
- Test breadcrumb links (relative paths)

### Mobile Layout Issues
- Test in actual devices (not just browser resize)
- Check viewport meta tag present
- Verify media queries in CSS

### Slow Load Times
- Optimize images (if added)
- Minify CSS/JS (optional)
- Enable browser caching
- Use CDN for Google Fonts
